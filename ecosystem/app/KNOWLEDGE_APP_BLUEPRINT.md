# KNOWLEDGE_APP_BLUEPRINT

## Objetivo
Planejar o futuro `predixai-knowledge-app`, uma aplicação real para memória, ideias, pesquisas, problemas, aulas, produtos, gráficos, mapas e galerias.

## Decisão
Não migrar agora. Criar primeiro um protótipo separado.

## Stack proposta
- React + Vite
- Firebase Auth
- Firestore
- Firebase Storage
- Firebase Hosting

## Telas iniciais
1. Login
2. Dashboard
3. Inbox da memória
4. Detalhe/editável da memória
5. Galeria
6. Gráficos
7. Exportação JSON
8. Sincronização com predixai-knowledge

## Regra
O app só vira oficial depois de validar UX, dados, login e custo.
