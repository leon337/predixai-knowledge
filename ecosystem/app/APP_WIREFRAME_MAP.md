# APP_WIREFRAME_MAP

## Objetivo
Mapear visualmente as telas do futuro predixai-knowledge-app antes de implementar código.

## Telas V1
1. Login
2. Dashboard
3. Inbox da Memória
4. Detalhe/Editável
5. Galeria
6. Relatórios
7. Configurações
8. Exportação/Sincronização

## Regra
Nenhuma tela deve ser implementada sem:
- objetivo;
- dados usados;
- ação principal;
- validação;
- saída esperada.
