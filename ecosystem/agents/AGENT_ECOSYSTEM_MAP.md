# AGENT ECOSYSTEM MAP

## Agentes principais
- ChatGPT: organização, pesquisa, explicação e documentação.
- Codex: execução supervisionada.
- OpenClaw: executor local controlado.
- Investigator: auditoria de evidências.
- Agentes futuros: comercial, conteúdo, curso, suporte e produto.

## Regra
Nenhum agente deve agir sem contexto, escopo, limite, validação e handoff.
