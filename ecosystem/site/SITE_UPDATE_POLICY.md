# SITE_UPDATE_POLICY

## Regra obrigatória
Toda atualização relevante em qualquer repositório oficial PredixAI BR deve gerar atualização correspondente no `predixai-knowledge`.

## O que atualizar
- Markdown de memória.
- Página HTML pública quando o conteúdo precisar ser consultado por humanos/agentes.
- JSON de contexto quando o estado do projeto mudar.

## Proibido
- Publicar código relevante sem atualizar memória.
- Criar PTP sem refletir decisão/contexto.
- Usar o site como decoração. O site é ferramenta operacional.
