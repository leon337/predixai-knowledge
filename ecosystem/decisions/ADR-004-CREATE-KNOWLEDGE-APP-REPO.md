# ADR-004 - Criar predixai-knowledge-app separado

## Status
Proposto.

## Decisao
Criar futuramente um repositorio separado chamado predixai-knowledge-app.

## Motivo
Reduz risco, preserva o portal atual e permite testar React/Firebase sem quebrar a memoria oficial.

## Nao fazer agora
- Nao migrar o predixai-knowledge.
- Nao abandonar GitHub Pages.
- Nao iniciar implementacao sem aprovacao.
