## 🎯 Objetivo

Estabelecer uma base sólida para que toda evolução da PredixAI siga uma metodologia única, consistente e versionada.

---

## 📌 Regras

- Todo novo projeto deve respeitar esta fundação.
- Alterações arquiteturais devem ser documentadas.
- O conhecimento deve ser reutilizável.
- O aprendizado deve ser incremental.
- O design deve seguir Mobile First, UI, UX e Learning Experience (LX).

---

> Toda evolução da PredixAI deve respeitar os princípios definidos nesta pasta.
