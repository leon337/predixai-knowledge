# Scripts

Pasta oficial de automações da PredixAI Knowledge Hub.

Esta pasta contém scripts PowerShell usados para criar, preencher, manter e automatizar a estrutura do repositório.

---

## Script principal

### bootstrap.ps1

Script oficial principal.

Responsável por:

- criar/verificar a estrutura de pastas;
- criar READMEs básicos;
- executar a geração completa dos documentos da Fundação;
- manter a base inicial da PredixAI Knowledge Hub reproduzível.

Uso:

```powershell
.\scripts\bootstrap.ps1
